# SmartHome-4U Design Grammar Pack v1

This package contains the machine-readable v1 foundation for the SmartHome-4U generative design engine.

## Included
- schemas/smarthome4u_archetype_schema_v1.json — master JSON schema
- schemas/space_dimension_library_v1.json — room and space standards
- schemas/rule_engine_pack_v1.json — universal design, planning, façade, LSF, MEP, and scoring rules
- templates/archetype_template_v1.json — archetype template
- templates/archetype_template_v1.yaml — human-editable template
- archetypes/json/ — 8 fully populated core archetypes in JSON
- archetypes/yaml/ — 8 fully populated core archetypes in YAML
